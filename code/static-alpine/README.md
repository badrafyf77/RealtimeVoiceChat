# Voice Chat Frontend

Alpine.js + Tailwind CSS + Vite frontend matching Wave SaaS design system.

## Quick Start

```bash
npm install
npm run build
```

## Development

```bash
npm run dev  # http://localhost:3000
```

## Tech Stack

- Alpine.js (reactive UI)
- Tailwind CSS (styling)
- Vite (bundler)

Outputs to `../static/` for FastAPI backend.
